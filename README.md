# Teslo API

1. Clone the repository
2. ```yarn install | yarn```
3. Clone the file ```.env.template``` and rename it to ```.env```
4. set the environment variables
5. Run de Database Up
```
docker-compose up -d
```
6. Run Seed 
```http://localhost:3000/api/seed
```
7. Run development server ```yarn start:dev```
